## Project Overview
This project builds a complete, production-ready data science pipeline designed to analyze the television e-commerce market. The pipeline automates data collection across dozens of pages, standardizes modern data engineering workflows by handling missing categorical and continuous values without deprecated syntax, and delivers advanced data visualizations. 

By analyzing the relationships between a TV's physical dimensions, display technologies, retail costs, and star scores, this pipeline uncovers what truly drives product pricing and customer satisfaction in the electronics marketplace.
